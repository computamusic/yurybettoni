## Responsive intent (every breakpoint is designed — doc 06 has the grid)

| Breakpoint | Behavior |
|---|---|
| **Desktop / iPad landscape (primary)** | Full layouts as drawn; multi-column pillar router, side-by-side media/text. |
| **iPad portrait** | 2-col → stacked; hero retains film; timeline becomes vertical. |
| **Mobile** | Single column; pillar router becomes a vertical stack; sticky buy bar on product/course; full-screen nav overlay; Subscribe pinned in footer. |

> Mobile is never an afterthought: each wireframe above has a defined stacked order, and conversion elements (CTAs, Subscribe) stay reachable without long scroll.
